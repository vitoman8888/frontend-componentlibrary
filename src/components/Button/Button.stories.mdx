# Button

The button component can be used by importing it into your app with the following code:

```js
    import { Button } from '@lernantino/up-design';
```

## Render a primary button

You can use a button for both DOM event binding or as a hyperlink &lt;a&gt;

<Preview withToolbar>
  <Story name='Primary'>
    <Button
      size='default'
      variant='primary'
      onClick={() => alert('this button was clicked')}
    >
      This is a primary button with a click event
    </Button>
  </Story>
</Preview>

## Render a secondary button

The large button can be used for accessibility.
